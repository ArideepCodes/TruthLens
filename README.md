# ✨ Overview

**TruthLens** is a futuristic AI vs Human text detection platform designed to analyze writing patterns using advanced heuristic-based forensic analysis entirely inside the browser.

The application estimates whether a piece of text is:
- human-written
- AI-generated
- suspiciously robotic
- behaviorally predictable

without relying on:
- APIs
- machine learning models
- backend servers
- cloud processing

TruthLens simulates real-world AI writing forensics through:
- burstiness scoring
- sentence structure inspection
- predictability analysis
- vocabulary diversity evaluation
- emotional variation detection

all powered using pure frontend technologies.
